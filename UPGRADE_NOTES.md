# Clwsd 升级笔记

## 1.0.0 功能

1. 从文件读取 URL
2. 逐个发起 HTTP/HTTPS 请求
3. 提取状态码、标题、响应长度、Server
4. 处理超时、连接失败、请求异常
5. 保存结果到输出文件

## 1.1.0 升级点

1. 读取目标时自动去重
2. 增加 `final_url`
3. 增加 `content_type`
4. 增加 `-a / --only-alive`，只输出存活目标
5. 输出字段更完整，便于后续分析

## 1.2.0 升级点

1. 增加 `-w / --workers` 参数
2. 使用 `ThreadPoolExecutor` 实现多线程并发探测
3. 使用 `as_completed()` 按完成顺序处理结果
4. 提高多目标探测速度
5. 增加线程任务异常捕获，避免单个任务影响整体运行

## 1.3.0 升级点

1. 增加 `-f / --format` 参数
2. 支持 TXT 输出
3. 支持 CSV 输出
4. 支持 JSON 输出
5. 为后续结构化分析做准备
